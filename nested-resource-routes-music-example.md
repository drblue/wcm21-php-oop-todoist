# Nested Resource Routes för musik-app

```text
/artists								# Visat lista över alla artister
/artists/42								# Visat info om artist #42
/artists/42/albums						# Visat alla album för artist #42
/artists/42/albums/1337					# Visat info om album #1337 som tillhör artist #42
/artists/42/albums/1337/tracks			# Visat alla låtar i album #1337 som tillhör artist #42
/artists/42/albums/1337/tracks/8192		# Visat info om låt #8192 för album #1337 som tillhör artist #42
```
